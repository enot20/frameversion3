# [Start Bootstrap - Grayscale](https://startbootstrap.com/theme/grayscale/)

[Grayscale](https://startbootstrap.com/theme/grayscale/) es un tema HTML de una sola página en escala de grises para [Bootstrap](https://getbootstrap.com/), creado por [Start Bootstrap](https://startbootstrap.com/).

## Inicio Rápido

```bash
# Clonar el repositorio
git clone https://github.com/StartBootstrap/startbootstrap-grayscale.git

# Entrar al directorio
cd startbootstrap-grayscale

# Instalar dependencias
npm install

# Iniciar el servidor de desarrollo
npm start
```

Abre `http://localhost:3000` en tu navegador para ver la vista previa en vivo.

## Instalación

Elige una de las siguientes opciones para comenzar:

- [Descargar la última versión en Start Bootstrap](https://startbootstrap.com/theme/grayscale/)
- Instalar via npm: `npm i startbootstrap-grayscale`
- Clonar el repositorio: `git clone https://github.com/StartBootstrap/startbootstrap-grayscale.git`
- [Hacer fork, clonar o descargar en GitHub](https://github.com/StartBootstrap/startbootstrap-grayscale)

## Compilación

Para compilar el proyecto para producción:

```bash
npm run build
```

La salida compilada se generará en el directorio `dist/`.

## Estructura del Proyecto

```
startbootstrap-grayscale/
├── dist/                    # Salida compilada (no editar directamente)
├── src/
│   ├── assets/              # Recursos estáticos (imágenes, fuentes)
│   ├── js/
│   │   └── scripts.js       # Archivos JavaScript
│   ├── pug/
│   │   ├── layouts/         # Plantillas base Pug
│   │   ├── pages/           # Plantillas de páginas
│   │   └── partials/        # Componentes reutilizables
│   └── scss/
│       ├── base/            # Estilos base
│       ├── components/      # Estilos de componentes
│       ├── layouts/         # Estilos de diseño
│       └── utilities/       # Clases utilitarias
├── scripts/                 # Scripts de compilación
├── package.json
└── README.md
```

## Scripts Disponibles

| Comando                 | Descripción                                                                |
| ----------------------- | -------------------------------------------------------------------------- |
| `npm start`             | Compila el proyecto e inicia el servidor de desarrollo con recarga en vivo |
| `npm run start:debug`   | Inicia el servidor de desarrollo en modo debug                             |
| `npm run build`         | Compila para producción (limpia + compila todos los recursos)              |
| `npm run build:pug`     | Compila plantillas Pug a HTML                                              |
| `npm run build:scss`    | Compila Sass/SCSS a CSS                                                    |
| `npm run build:scripts` | Procesa archivos JavaScript                                                |
| `npm run build:assets`  | Copia recursos estáticos a dist                                            |
| `npm run clean`         | Elimina el directorio dist                                                 |

## Características

- **Bootstrap 5.2.3** - Sistema de rejilla responsive y componentes
- **Plantillas Pug** - HTML mantenible con componentes reutilizables
- **Sass/SCSS** - Estilos personalizados con variables y mixins
- **BrowserSync** - Recarga en vivo y pruebas sincronizadas
- **Autoprefixer** - Prefijos de proveedor automáticos para CSS
- **Prettier** - Formateo de código
